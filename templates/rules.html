📖 Rules & Guidelines

⏱️

Time Limit

Each word has a time limit based on its length and difficulty level. If time runs out, you'll lose points and the word resets.

⌨️

Virtual Keyboard

The on-screen keyboard shows which keys you should press. Active keys are highlighted in green, wrong keys in red.

⭐

Scoring System

  • ✅ Correct word: +3 points per character
  • ❌ Wrong word: -5 points
  • 🔑 Adjacent key error: -4 extra points
  • ⏰ Time's up: -5 points
🔒

Sequential Lock

You must complete lessons in order. Each lesson must be finished before unlocking the next one.

🌍

Adding New Languages

To add a new language, create a JSON file in the lessons/ folder with this format:

{
  "name": "Language Name",
  "code": "lang_code",
  "lessons": [
    {
      "id": 1,
      "title": "Lesson Title",
      "subtitle": "Subtitle",
      "words": ["word1", "word2", "word3"],
      "hint_keys": ["a", "s", "d", "f"]
    }
  ]
}

📁 File name: language_code.json (e.g., spanish.json)

💡 The language will automatically appear in the login page!

🏆

Mastery Goal

Complete all lessons with high accuracy and speed to become a professional typist!
